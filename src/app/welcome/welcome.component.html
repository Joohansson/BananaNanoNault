  • Why Use BananoNanoNault?
    Focused on Security
    Specifically made for accessing Banano locked by a Ledger for Banano accounts maching Nano accounts. Usually when airdrops happens. Banano can then be accessed by connected the Ledger using the Nano app (not banano).
    This app is using the Banano network. Another option is to use remote signing via an offline device in order to protect your keys.
    Any Device, Anywhere
    Access banano via the web, or download the desktop wallet available for Windows/Mac/Linux.
BananoNanoNault is open source and all code is available at https://github.com/Joohansson/BananoNanoNault.
